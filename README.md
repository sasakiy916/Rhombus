# ひし形の図形表示
## お題
奇数を入力するとひし形を表示するようにしてくだい。 
## 仕様条件
1. 画面に表示する 
奇数を入力してください>>(ここに任意の値を入力するようにする) 
2. 奇数以外が入力されたら再入力をするようにする
## 実行例 
好きな奇数を入れてください>>5
  X

  X
　

好きな奇数を入れてください>>8
奇数以外が入力されました。再入力してください。
